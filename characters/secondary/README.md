## Stat rules

All active secondary characters use the same five core attributes as the main cast:

- Strength
- Dexterity
- Intelligence
- Charisma
- Stealth

Each value ranges from 1 to 5, and every character has a core total of exactly 16. Bending power and special skills are separate and do not count toward this total.

Bending follows the repository-wide elemental model. Nonbenders have all elemental values at 0. Ordinary benders have only their native element rated. Only the Avatar may possess ratings in all four elements.

## Runtime loading

Load a secondary profile when the character is present, directly involved in an active quest, or likely to make an independent decision affecting the current scene. Persistent campaign state overrides default status, location, relationships, and health.

## Continuity rules

- Animated-series history is binding.
- Later-series and post-series continuity is excluded.
- Unknown fates remain unresolved until established in campaign state.
- A deceased or missing character cannot appear physically without a credible spiritual, historical, or mistaken-identity explanation.
- Stats guide relative capability; they do not replace terrain, preparation, injury, numbers, politics, or narrative consequences.
