# Foundation Reading Index

Load these files for every new campaign or major simulation reset:

1. `../SYSTEM_PROMPT.md`
2. `canon_boundary.md`
3. `history.md`
4. `timeline.md`
5. `world_state.md`
6. `politics.md`

## Purpose of Each File

- `history.md` explains the historical forces that created the current world.
- `timeline.md` provides chronological continuity and the original six-year bridge.
- `world_state.md` defines the campaign-start conditions and active pressures.
- `politics.md` defines institutions, factions, interests, and conflict-generation rules.

## Retrieval Rule

Do not load every repository file into every scene. Load the foundation files, then add the current nation, location, active factions, involved characters, and relevant gameplay systems.
