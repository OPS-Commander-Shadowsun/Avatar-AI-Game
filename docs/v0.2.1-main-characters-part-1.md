# v0.2.1 — Main Characters Part 1

## Scope

This milestone expands the first five core characters into AI-ready simulation profiles:

- Aang
- Katara
- Sokka
- Fire Lord Zuko
- Toph Beifong

## Improvements

Each profile now contains a detailed visual description, psychology, core values, speech patterns, canonical biography, postwar role, abilities, combat behavior, relationships, adult romance and family expectations, daily patterns, knowledge limits, current objectives, corruption stages, redemption routes, portrayal constraints, and quest hooks.

## Design goals

- Preserve recognizable characterization without copying scripts or dialogue.
- Separate animated canon from the game-original six-year bridge.
- Prevent NPCs from becoming omniscient, instantly romanceable, or instantly corruptible.
- Give the narrator concrete constraints for blindness, bending, political authority, trauma, travel, and personal obligations.
- Support dynamic relationships while respecting existing default commitments.

## Branch strategy

All later milestones should continue on `agent/main-characters-v0.2.1` so the final branch contains the complete accumulated World Bible.
